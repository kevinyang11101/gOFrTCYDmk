# 二手房管理系统 java1174

## 项目概述

二手房管理系统是一款基于SSM框架和Vue.js的前后端分离的房产信息管理系统。系统致力于提供高效、稳定的服务，以适应不同的业务需求。

## 技术栈

### 前端

- **Vue.js**：用于构建动态交互界面。

### 后端

- **Spring**：提供后端的业务逻辑处理。
- **MyBatis**：数据持久化框架。

### 数据库

- **MySQL 5.x/8.x**：支持双版本适配，确保数据存储的可靠性。

### 运行环境

- **JDK 8**：Java开发环境。
- **Maven**：项目管理和构建自动化工具。

## 功能模块

系统包含但不限于以下功能模块：

- 用户管理
- 房源管理
- 合同管理
- 费用管理
- 数据报表

## 系统角色

- 系统管理员
- 员工
- 客户

## 部署步骤

1. 环境准备：确保JDK 8和Maven已安装。
2. 数据库配置：根据提供的数据库脚本初始化MySQL数据库。
3. 后端部署：通过Maven构建项目，部署到相应的服务器。
4. 前端部署：编译Vue.js项目，部署到Web服务器。
